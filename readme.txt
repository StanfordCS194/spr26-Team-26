Team Nemoral

PROJECT: Autonomous ML Training Agent

Give it a plain-English prompt (e.g. "build a model that classifies handwritten
digits") and a hard budget cap (e.g. $50), and the agent handles the rest:
finds or generates data, picks a training strategy (LoRA fine-tune or custom
pre-train), runs an AutoResearch loop on Tinker's distributed GPUs, and shuts
down before burning through your wallet.
